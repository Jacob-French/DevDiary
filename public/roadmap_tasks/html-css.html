What is a HTML element?

An element is everything from a start tag to an end tag including the tag contents.

What is a selector?

A selector selects HTML elements to be styled. For example the element selector, id selector and class selector.

What are the components of the box model?

Margin, border, padding, content

What is a combinator?

A combinator can be added to a selector to make a more specific selection. > + ~ characters can be placed between selectors to select for children and siblings.

What is a pseudo class?

A pseudo class is used to define a special state of an element. For example, the hover pseudo class div:hover{}
